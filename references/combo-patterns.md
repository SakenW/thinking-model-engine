# 经典模型组合配方

> 经过验证的多模型协同分析模式。每个配方标注了适用场景和实战记录。

---

## 配方 1: 🎯 个人重大决策

**场景：** 跳槽、买房、创业、结婚、移民等人生重大选择

```
第一步：全景扫描
  SWOT（内外因素）+ 黄金圈（价值观对齐）

第二步：量化对比
  决策矩阵（多准则打分）+ 机会成本（真实代价）

第三步：防偏误检查
  沉没成本（已投入的不算）+ 锚定效应（第一印象不锚定）
  + 前景理论/损失厌恶（不怕亏就怕错失）

第四步：风险深挖
  逆向思维（怎么让这个决定失败）+ 二阶思维（后果的后果）

第五步：长期校验
  10-10-10 旁观思维 + 路径依赖评估
```

**输出格式：** 决策评分表 + 风险清单 + 行动计划

## 配方 2: ⚡ 效率与自律系统重建

**场景：** 总是拖延、任务堆积、感觉忙但没产出

```
第一步：诊断
  艾森豪威尔矩阵（当前任务分布）+ 帕金森定律检查（是否在膨胀）
  + 帕累托法则（找到高杠杆事项）

第二步：行为设计
  福格 B=MAP（让行动更容易触发）+ WOOP（识别真正障碍）

第三步：执行系统
  GTD 完整流程（收集→澄清→组织→回顾→执行）+ 时间阻塞（深度工作时段）

第四步：持续优化
  PDCA 循环（每周迭代改进）+ 反馈回路（建立进度感知）
```

**输出格式：** 当前状态诊断 + 新系统设计 + 第一周执行计划

## 配方 3: 📚 快速学习新领域

**场景：** 需要在短时间内掌握一个新技能/知识领域

```
第一步：定位
  达克效应自评（我在哪？愚昧之山还是绝望之谷？）

第二步：输入
  SQRRR 阅读法（主动阅读所有材料）

第三步：消化
  费曼学习法（用简单语言解释给12岁孩子听）
  + 金字塔原理（把知识结构化成体系）

第四步：巩固
  蔡格尼克记忆效应利用（故意留尾巴明天继续）
  + 元认知监控（观察自己的理解过程）

第五步：输出
  SCQA 或 PREP 结构化表达（能讲清楚才算真会）
```

**输出格式：** 学习路径图 + 费曼式通俗解释 + 知识体系图

## 配方 4: 💼 产品/业务增长突破

**场景：** 用户增长停滞、收入下滑、需要找到突破口

```
第一步：数据诊断
  销售漏斗（转化率逐层分析）+ AISAS（用户旅程完整路径）

第二步：根因挖掘
  5 Whys（为什么增长不动？追问5层）

第三步：机会发现
  需求月牙锯（凹陷区=被忽视的机会）+ BCG矩阵（资源重新分配）

第四步：方案设计
  设计思维（从用户痛点出发）+ 价值主张画布（产品-市场匹配）

第五步：执行规划
  MVP 最小可行产品（低成本验证）+ PDCA（快速迭代）
```

**输出格式：** 诊断报告 + 机会清单 + MVP 方案 + 90天路线图

## 配方 5: 😤 冲突沟通 / 高难度对话

**场景：** 和老板/客户/伴侣有分歧，需要有效沟通

```
第一步：自我准备
  非SR思维模型（先暂停，不自动反应）+ 乔哈里视窗（我的盲区在哪？）

第二步：理解对方
  马斯洛需求层次（他真正的需求是什么？）+ 贝勃定律（他的感知阈值如何？）

第三步：构思表达
  SCQA（情境→冲突→问题→答案的叙事）+ PREP（精简版论点）
  + FFC 赞美法（先建立连接再谈分歧）

第四步：对话中
  六顶思考帽（切换角度，尤其是对方的"红帽"情绪）
  + 五遍沟通法（确认信息准确传递）

第五步：后续跟进
  反馈回路（建立持续沟通机制）
```

**输出格式：** 对话 prep 清单 + 关键话术 + 风险预案

## 配方 6: 🔬 复杂问题的系统性复盘

**场景：** 项目结束、季度/年度总结、失败后的深度反思

```
第一步：事实回顾
  GRAI（Goal→Result→Analysis→Insight）或 KISS（Keep/Improve/Start/Stop）

第二步：根因深挖
  5 Whys（表层原因→根本原因）+ 大冰山模型（事件→模式→结构→心智模型）

第三步：系统洞察
  Cynefin 框架（这类问题属于什么类型？方法对了吗？）
  + 反馈回路（正/负反馈是否在起作用？）

第四步：知识沉淀
  SECI（隐性经验→显性知识）+ 费曼学习法（能把经验教给别人吗？）

第五步：改进闭环
  PDCA（下一轮的计划→执行→检查→改进）
```

**输出格式：** 复盘报告 + 根因树 + 改进 action items + 知识沉淀

## 配方 7: 🚀 创业 / 新项目从0到1

**场景：** 准备开始一个新项目、新产品、新业务

```
第一步：初心与方向
  黄金圈法则（Why→How→What）+ OGSM（战略→目标→策略→度量）

第二步：市场与环境
  PESTEL（宏观六力）+ 波特五力（行业竞争格局）+ STP（细分→目标→定位）

第三步：商业模式
  商业模式画布（9要素完整设计）+ 价值主张画布（产品-市场匹配）

第四步：风险与韧性
  黑天鹅（极端风险）+ 安全边际（缓冲空间）+ 路径依赖（锁定风险评估）

第五步：执行启动
  MVP（最小可行验证）+ PDCA（快速迭代循环）+ OKR（团队目标对齐）
```

**输出格式：** 商业模式画布 + 风险矩阵 + MVP 定义 + Q1 OKR
