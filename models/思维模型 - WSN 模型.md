# WSN 模型 (What / So What / Now What)

## 概述

**三层递进式反思框架**，从"发生了什么"到"意味着什么"再到"接下来做什么"。广泛用于复盘、反思、引导讨论。

## 三层结构

| 层次 | 问题 | 产出 |
|------|------|------|
| **What** | 发生了什么？事实是什么？ | 客观描述，不带评判 |
| **So What** | 那又怎样？为什么重要？有什么影响？ | 分析意义，提炼洞察 |
| **Now What** | 现在怎么办？下一步做什么？ | 行动计划，具体行动 |

## 示例：项目复盘

### What（发生了什么）
- 项目延期了 2 周
- 测试阶段发现了 47 个 bug
- 团队有 3 个人中途被抽调去支持其他项目
- 需求在开发过程中变更了 8 次

### So What（那又怎样）
- 延期的根因不是技术问题，而是**资源不稳定 + 需求蔓延**
- 47 个 bug 中 60% 来自需求不明确导致的返工
- 团队士气受影响，2 名核心成员表示压力过大
- **核心洞察：项目启动时需求锁定和资源承诺机制缺失**

### Now What（现在怎么办）
1. 立即：建立需求变更审批流程（需 PM + Tech Lead 双签）
2. 本周：与资源管理者确认团队稳定性承诺
3. 下个项目启动前：引入需求评审会（含签字确认环节）
4. 每月：做一次团队健康度检查

## 适用场景

- 项目 / 活动 / 事件复盘
- 学习反思（读完一本书/上完一门课）
- 会议总结
- 年度/季度回顾
- 引导式讨论和教练对话

## 使用技巧

- **What 阶段要克制** — 只说事实，不要跳到分析
- **So What 是关键** — 很多人容易跳过这一层直接给方案
- **Now What 要可执行** — 不要写"下次注意"，要写具体动作
- 可以个人用，也可以团队一起填

## 变体

### What → If What → Then What
用于情景推演：
- What: 当前情况
- If What: 如果…会怎样？
- Then What: 那么…

### 4F (Facts, Feelings, Findings, Futures)
更偏体验和学习场景。
