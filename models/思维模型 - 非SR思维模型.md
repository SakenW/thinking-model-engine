# 非SR思维模型 (Non-SR Thinking)

## 概述

**SR = Stimulus-Response（刺激-反应）**，指外界刺激直接触发自动反应的模式。**非 SR 思维就是在刺激和反应之间插入一个"间隙"——在这个间隙里选择如何回应，而非被本能/习惯/情绪劫持。**

## 核心公式

```
传统模式 (SR)：
  刺激 → 反应
  （有人骂我 → 我立刻骂回去）

非 SR 模式：
  刺激 → [间隙/觉察] → 选择 → 回应
  （有人骂我 → 等等，我为什么想骂回去？→ 这是我的课题吗？→ 冷静回应）
```

## 维克多·弗兰克尔的名言

> "在刺激和回应之间有一个空间。在这个空间里，我们自由选择我们的回应。在我们的回应中，蕴藏着我们的成长和自由。"
> 
> — 维克多·弗兰克尔，《活出意义来》

## 三层练习

| 层次 | 练习 | 效果 |
|------|------|------|
| **L1: 物理暂停** | 刺激后深呼吸 3 次 / 数到 10 | 打断自动反应链 |
| **L2: 认知重评** | 问自己："这是事实还是我的解读？""对方为什么这么做？" | 改变对刺激的理解 |
| **L3: 主动选择** | 基于价值观而非情绪选择回应方式 | 最高级的自由 |

## 使用步骤

1. **识别你的 SR 触发器** — 什么最容易让你"自动反应"？（被批评、被忽视、不公平…）
2. **设计你的"暂停信号"** — 一个物理动作或心理暗示（摸戒指、深呼吸、心里说"pause"）
3. **在低风险场景练习** — 不要等大危机才第一次用
4. **事后复盘** — 这次成功插入间隙了吗？哪里做得好？
5. **逐步升级** — 从小事开始，逐步应用到高压力场景

## 示例

> **收到一封愤怒的邮件：**
>
> - **SR 反应**：立刻回怼，证明自己是错的…写了一封更愤怒的回复…
> - **非 SR 反应**：
>   - 🛑 暂停：不立刻回复，放下手机
>   - 💭 重评：他为什么生气？是不是有误会？他的攻击是针对我还是针对事情？
>   - ✅ 选择：等明天早上冷静后再回，只针对事实不针对情绪
>
> **看到令人焦虑的新闻：**
>
> - **SR**：立刻转发 + 评论 + 焦虑一整天
> - **非 SR**：停一下——这条新闻对我有直接影响吗？我能做什么？转发能改变什么？

## 注意事项

- 非 SR ≠ 不回应或压抑情绪——而是**有意识地选择**如何回应
- 间隙不需要很长——**几秒钟就够**，关键是"有"这个间隙
- 高压力下间隙会变短——**平时练习才能在关键时刻用上**
- 这不是一次学会的技能——**终身练习的过程**

## 变体/相关模型

- **斯多葛哲学 (Stoicism)**：非 SR 思想的哲学根源
- **情商 (EQ / Emotional Intelligence)**：Goleman 的框架——自我调节是非 SR 的核心能力
- **ACT 接受承诺疗法**：心理治疗中的类似概念——"defusion"（认知解离）
- **反应性 vs 主动性 (Reactive vs Proactive)**：《高效能人士的七个习惯》中的概念
- **元认知**：非 SR 的底层能力——观察自己的思考过程

## 适用场景

- 冲突沟通（家庭、职场、网络）
- 情绪管理（愤怒、焦虑、冲动消费）
- 危机应对（坏消息、突发事件）
- 习惯改变（打断坏习惯的自动触发）
- 谈判与博弈
- 日常决策质量提升
- 人际关系维护
