# 洛克目标设定理论 (Locke's Goal-Setting Theory)

## 概述

**明确且有挑战性的目标，比模糊或容易的目标，更能带来高绩效。**

埃德温·洛克（Edwin Locke）1968 年提出，是组织心理学中被实证支持最多的理论之一。

## 五个核心原则

| 原则 | 内容 | 关键点 |
|------|------|--------|
| **明确性** | 目标要具体可衡量 | "提升销售额 15%" > "做得更好" |
| **挑战性** | 目标要有难度但可达 | 太容易没动力，太难会放弃 |
| **承诺度** | 个人认同并承诺这个目标 | 自主设定的目标比被分配的更有动力 |
| **反馈** | 持续获得进度信息 | 不知道进度等于没有目标 |
| **任务复杂性** | 复杂任务需要更多时间、更细拆分 | 不能一个目标涵盖太多 |

## 核心发现

```
绩效
  ↑
  │                    ╱
  │                  ╱
  │                ╱   ← 最优点（有挑战但可达）
  │              ╱
  │            ╱
  │          ╱
  │        ╱
  │      ╱
  │    ╱
  └──────────────────────→ 目标难度
  模糊    明确    适中    过难
```

### 量化发现
- 明确目标比模糊目标平均提升绩效 **16%**
- 有挑战性目标比容易目标提升更多
- 但**过难**的目标反而导致绩效下降（挫败感）

## 与 SMART 原则的关系

洛克理论是 SMART 原则的**学术基础**：

| SMART 维度 | 对应洛克原则 |
|-----------|-------------|
| Specific | 明确性 |
| Measurable | 明确性 |
| Achievable | 挑战性（可达） |
| Relevant | 承诺度 |
| Time-bound | 反馈周期 |

## 实际应用

### 设定目标时
1. 写下来（具体可衡量）
2. 确保有挑战性但不是不可能
3. 自己认同（不是纯被动接受）
4. 设定检查点（周/月反馈）

### 管理团队时
- 参与式目标设定 > 自上而下指派
- 公开目标比私下目标更有约束力
- 复杂项目拆成子目标，每个子目标都遵循五原则

## 注意事项

- **不适用于所有任务**：创造性/探索性工作，过明确的目标可能限制发散思维
- **学习阶段不宜太难**：新手需要先建立信心（配合达克效应理解）
- 和 OKR 配合使用：OKR 的 KR 就是洛克理论的"明确+可衡量"落地

## 与其他模型的关系

- **SMART 原则**：洛克理论的实践版本
- **OKR**：组织级目标管理的洛克理论应用
- **心流**：挑战性与技能匹配 → 心流状态
- **达克效应**：能力低时自设目标容易偏差
