# 艾森豪威尔矩阵 (Eisenhower Matrix)

## 概述

按 **重要** 和 **紧急** 两个维度，把任务分为四象限，决定做、排期、委托还是删除。

据传源自德怀特·艾森豪威尔（美国第 34 任总统）的时间管理原则。

## 四象限

```
          紧急 ↑
    ┌───────────┼───────────┐
    │           │           │
    │  🔥 第一   │  🔄 第二  │
    │  象限      │  象限     │
    │  做！      │  规划     │
    │ (重要+紧急)│(重要不紧急)│
    ├───────────┼───────────┤→ 重要
    │           │           │
    │  🗑️ 第三   │  ❌ 第四  │
    │  象限      │  象限     │
    │  委托/减少  │  删除    │
    │(紧急不重要)│(不重要    │
    │           │ 不紧急)   │
    └───────────┴───────────┘
        不重要 ←——————— 重要 →
```

## 各象限策略

### 🔥 第一象限：重要 + 紧急 → **立刻做**
- 危机处理、截止日期临近的任务、突发问题
- **目标：尽量缩小这个区域**
- 很多"紧急"是因为第二象限没做好

### 🔄 第二象限：重要 + 不紧急 → **规划安排（核心！）**
- 长期规划、能力建设、关系维护、健康锻炼
- **这是决定人生质量的关键区域**
- 乔布斯："我的一天从最重要的事开始"
- **策略：主动为它分配固定时间**

### 🗑️ 第三象限：不重要 + 紧急 → **委托 / 减少打断**
- 大部分会议、突发邮件、别人的紧急事
- **策略**：
  - 学会说"不"或"稍后"
  - 委托给合适的人
  - 批量处理邮件和消息

### ❌ 第四象限：不重要 + 不紧急 → **删除**
- 无意义的刷手机、无目的的闲聊、逃避性活动
- **策略：直接砍掉**

## 实用技巧

### 判断标准
| 维度 | 问自己 |
|------|--------|
| 重要吗？ | 做了这件事，5 年后还有影响吗？ |
| 紧急吗？ | 今天不做会有可见的负面后果吗？ |

### 常见误区
- 把"别人觉得紧急"当成"紧急"
- 把"我喜欢做的事"自动归入"重要"
- 第二象限永远被第一象限挤占 — **必须保护第二象限时间**

### 时间建议分配
- 第一象限：10-15%
- 第二象限：**60-70%**（应该最多）
- 第三象限：15-20%
- 第四象限：< 1%（尽量趋近于 0）
