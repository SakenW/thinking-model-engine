# 曼陀罗思考法 (Mandala Chart / 9-Window Matrix)

## 概述

源自日本的系统化思考工具，用 **3×3 的九宫格**（曼陀罗）来展开思维。从一个中心主题出发，向 8 个方向发散，每个方向还可以继续展开成新的九宫格。**让平面思考变立体，线性思考变网状。**

## 基本结构

```
┌─────┬─────┬─────┐
│  1  │  2  │  3  │
│     │     │     │
├─────┼─────┼─────┤
│  4  │ ★  │  6  │   ★ = 核心主题
│     │中心 │     │
├─────┼─────┼─────┤
│  7  │  8  │  9  │
│     │     │     │
└─────┴─────┴─────┘
```

## 三种用法

### 用法一：发散思维（8 方向展开）

中心写主题，8 个格子填 8 个相关方向。

```
        ┌───┬───┬───┐
        │人 │财 │物 │
        ├───┼───┼───┤
        │时 │项目│技 │
        ├───┼───┼───┤
        │地 │信 │组 │
        └───┴───┴───┘
```

### 用法二：时间管理（今日曼陀罗）

把一天的重要事项放入 9 格：

| | 早 | 中 | 晚 |
|--|----|----|----|
| **重要** | 深度工作 | 关键会议 | 复盘/学习 |
| **普通** | 邮件处理 | 协作沟通 | 家务/运动 |
| **生活** | 早餐/冥想 | 午休/社交 | 家庭时间 |

### 用法三：目标拆解（曼陀罗套娃）

中心 = 大目标 → 8 个子目标 → 每个子目标再展开 9 格

## 使用步骤

1. **在中心写下核心问题/主题**
2. **快速填充 8 个方向** — 不要思考太久，先填满
3. **审视是否有遗漏** — 8 个方向覆盖全面吗？
4. **选择重要的方向继续展开** — 做第二层曼陀罗
5. **整合输出** — 从曼陀罗中提炼出行动计划

## 示例

> **"如何提升个人影响力"的曼陀罗：**
>
> - 中心：个人影响力
> - ①写作输出 ②公开演讲 ③社交媒体 ④专业深度 ⑤人际网络 ⑥个人品牌 ⑦视觉形象 ⑧帮助他人
>
> 选择"写作输出"继续展开：
> - 中心：写作输出
> - ①公众号 ②技术博客 ③知乎/专栏 ④书籍 ⑤Newsletter ⑥短视频脚本 ⑦内部文档 ⑧日记/反思

## 注意事项

- 第一层 8 个格子的质量决定了后续展开的价值——**不要随便填**
- 不是所有方向都值得展开——**聚焦最重要的 2-3 个**
- 曼陀罗可以和思维导图配合使用——曼陀罗做结构，导图做细节
- 8 个方向不一定要"均匀"——有些方向天然内容更多

## 变体/相关模型

- **思维导图 (Mind Map)**：更自由的发散工具，没有 8 方向限制
- **MECE 原则**：确保 8 个方向不重不漏的方法论
- **SCAMPER**：创意生成的七步法，可放入曼陀罗的一个方向
- **六顶思考帽**：曼陀罗中的某个方向的深入分析工具
- **概念地图 (Concept Map)**：学术版的关联思考工具

## 适用场景

- 头脑风暴与创意生成
- 项目规划与任务拆解
- 目标分解（年度计划 → 季度 → 月度）
- 问题分析（多维度扫描）
- 学习笔记整理（一章书的 8 个要点）
- 时间管理（每日/每周安排）
