# 赫兹伯格双因素理论 (Herzberg's Two-Factor Theory)

## 概述

**影响工作满意度的因素分两类：保健因素（不满足会不满，满足了也不会更满意）和激励因素（真正驱动满意和动力）。**

弗雷德里克·赫兹伯格（Frederick Herzberg）1959 年提出，又称"激励-保健理论"。

## 两类因素

```
满意度
  ↑
  │            ╱ 激励区
  │          ╱
  │        ╱
  │──────┼──────────────────
  │      ╱   ╲
  │    ╱      ╲ 不满区
  │  ╱          ╲
  └──────────────────────→
       保健因素不足     激励因素充足
```

| 类型 | 因素 | 特征 | 类比 |
|------|------|------|------|
| **保健因素** | 薪资、工作环境、公司政策、人际关系、工作稳定性 | 不满足 → 不满；满足了 → **不会**更积极 | 治病：治好了≠健康 |
| **激励因素** | 成就感、认可、工作本身、责任感、成长空间 | 满足了 → **真正的满意和动力** | 保健：超越了才有活力 |

## 核心洞察

> **加薪只能消除不满，不能创造动力。真正的动力来自成就感、认可和成长。**

### 常见误区

| 误区 | 真相 |
|------|------|
| "钱给够了就有动力" | 钱够了只是不走了，不代表积极 |
| "改善办公环境能提升士气" | 环境好了只是不抱怨了 |
| "福利好=员工满意" | 福利是保健因素，不是激励因素 |

## 实际应用

### 管理者
1. **先确保保健因素达标** — 薪资公平、环境基本舒适、政策合理
2. **再投入激励因素** — 给有挑战性的任务、及时认可、创造成长空间
3. **不要用保健因素替代激励因素** — 加薪不能代替"你的工作很有意义"

### 个人
- 选工作时：不只看薪资（保健），更要看成长空间和成就感（激励）
- 跳槽原因：如果只是薪资不满，可能加薪就能解决；如果缺乏成就感，换公司也不一定好

### 产品设计
- 基础功能 = 保健因素（没有用户走，有了不加分）
- 惊喜体验 = 激励因素（让用户真正爱上产品）

## 与马斯洛需求层次的关系

```
马斯洛          赫兹伯格
─────────      ─────────
自我实现  ←→   激励因素
尊重需求  ←→   激励因素
归属需求  ←→   保健/激励（边界）
安全需求  ←→   保健因素
生理需求  ←→   保健因素
```

## 注意事项

- 两类因素的**边界因人而异**（对新人，薪资可能是激励因素）
- 文化差异：不同文化对因素的敏感度不同
- 不意味着保健因素不重要——它是不满的"底线"
