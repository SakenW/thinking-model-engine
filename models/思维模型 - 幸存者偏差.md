# 幸存者偏差 (Survivorship Bias)

## 概述

**我们只看得见活下来的样本,看不见死掉的**。当某种筛选机制让"成功者"被看见、"失败者"被遗忘或隐藏时,基于可见样本得出的结论就会系统性偏颇——你以为找到了成功规律,其实只是找到了"幸存的偶然"。

经典源头:二战时美军统计返航战斗机上的弹孔分布,打算加固中弹多的部位。统计学家 Abraham Wald 反过来指出——**这些飞机能飞回来,说明它们中弹的部位不致命;真正致命的是那些没飞回来的飞机的中弹部位**。结论应该是加固返航机上**没有弹孔**的地方(引擎、驾驶舱)。

## 核心机制

```
    全部样本(N)
    ┌──────────────┐
    │              │
    │  淘汰/沉默    │ ←── 看不见的数据(最关键的真相往往在这)
    │  (失败者)     │
    ├──────────────┤
    │  被看见的样本  │ ←── 我们据此推理(但样本是筛选过的)
    │  (幸存者)     │
    └──────────────┘
    
    筛选机制 = 战争击落 / 市场淘汰 / 算法不推荐 / 自我隐藏
```

只要存在以下任一情况,幸存者偏差就可能出现:
- **筛选机制** — 只有"过线"的样本能进入视野
- **自报偏差** — 成功者愿意讲,失败者不愿讲
- **遗忘效应** — 时间让失败的案例被淡忘
- **可见度差异** — 失败的产品/公司连记录都没留下

## 操作步骤

1. **识别样本来源** — 我看到的这些案例是怎么被"挑出来"的?有没有筛选机制?
2. **追问沉默样本** — 那些没出现的样本去哪了?为什么消失?
3. **想象反事实** — 如果失败者也被记录下来,他们会告诉我什么?
4. **主动寻找失败案例** — 搜"为什么 X 失败了"而不是"X 怎么成功的"
5. **用全样本数据** — 找包含成功+失败的完整数据集,而不是只看胜出者

## 示例

> **成功学的陷阱:**
>
> - 书店满满都是"乔布斯/马斯克/比尔·盖茨的 10 个习惯"
> - 但有数以万计拥有完全相同习惯的人**破产了**,他们没出书,你看不到
> - 真相:这些习惯 ≠ 成功,只是幸存者的共同点。决定成败的可能恰恰是看不见的运气、时机、家世
>
> **创业经验分享:**
>
> - 一位连续创业者讲"我三次失败的经验,总结就是没找到 PMF"
> - 问题:能讲经验的人至少活到了能复盘的阶段。**最该听的失败者是那些再也没机会复盘的人**
>
> **客户调研:**
>
> - "我们调研了 500 位付费用户,满意度 95%!"
> - 沉默的样本是谁?——**流失的用户、试用后没买的、根本不知道你产品的人**
> - 真相往往藏在流失用户和潜在用户的访谈里
>
> **历史上的医疗:**
>
> - "古代用这个偏方的人都活下来了"——那些没活下来的人不会说话
> - 没有对照组的"疗效见证",几乎都是幸存者偏差

## 注意事项

- **成功者会反向归因** — 成功者倾向于把自己的成功归结为能力(而非运气),他们的"经验总结"本身就带着偏差
- **失败者的数据最难找** — 破产的公司网站都没了,失败的项目文档被删,这是结构性的信息缺失,需要主动挖掘
- **大数也不能解决** — 1000 个幸存者的样本,本质还是幸存者,**不能抵消偏差**
- **算法时代更严重** — 推荐算法会持续给你"成功的同类",让你以为这就是世界的全部
- **不要走到反面** — 警惕偏差 ≠ "成功都是运气"——规律可能真实存在,只是不能仅凭幸存样本推断

## 变体/相关模型

- **选择偏差 (Selection Bias)** — 更广义的样本筛选误差,幸存者偏差是其中一种
- **发表偏差 (Publication Bias)** — 学术界:发表的研究偏向有显著结果的研究,阴性结果被埋没
- **幸存者偏差的反面 = 失败者偏差** — 只看失败也可能误判(失败的偶然 ≠ 必然)
- **回归均值** — 极端成功者下一代往往平庸,部分原因是"成功本身有运气成分"
- **黑天鹅理论** — 沉默的失败样本中可能藏着极端事件,看不到不代表不存在

## 适用场景

- 商业决策(避免只学头部公司的成功经验)
- 创业与投资(找失败案例做对照)
- 用户/客户调研(必须访谈流失和非用户)
- 人才选拔(避免只看在职绩优者的画像)
- 学习方法选择(避免只学高分考生,要看中等生为什么没上去)
- 医疗与养生(警惕" testimonial"式证据)
- 历史分析(避免被传世文献带偏,要还原失声群体)
