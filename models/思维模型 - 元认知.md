# 元认知 (Metacognition)

## 概述

**"关于思考的思考"**——对自己认知过程的觉察、监控和调节。简单说就是：**你不仅在思考，还在观察自己怎么思考**。元认知是学习能力的底层操作系统，决定了你能多快掌握新技能。

## 两个层次

| 层次 | 含义 | 问题 |
|------|------|------|
| **元认知知识** | 知道自己的思维模式 | "我擅长什么类型的思考？我的认知偏差有哪些？" |
| **元认知调节** | 实时监控和调整思维过程 | "我现在是不是在走神？这个推理有没有漏洞？要不要换个思路？" |

## 元认知的三个核心动作

```
┌─────────────────────────────────────┐
│           元认知循环                 │
│                                     │
│   ① 计划 → "我要怎么做这件事？"     │──┐
│             │                       │  │
│   ② 监控 ← "我做得怎么样？"        │←─┘
│             │                       │
│   ③ 评估 → "结果如何？下次怎么改？" │──→ 下一次计划
└─────────────────────────────────────┘
```

## 使用步骤

### 日常练习方法

1. **思维标签化** — 思考时给自己打标签："我在做归纳""我在做类比""我在情绪化判断"
2. **暂停与抽离** — 遇到难题时停 5 秒问："我现在的思路对吗？"
3. **思维日记** — 每天记录一个"我今天是怎么想的"案例
4. **事后复盘** — 做完决策后回顾：当时为什么这么想？哪里想错了？
5. **故意换视角** — 强制自己用反对者的立场重新审视结论

## 示例

> **阅读时的元认知：**
>
> - 普通阅读：从头看到尾，看完"看懂了"
> - 元认知阅读：
>   - **计划**："这篇文章我想了解什么？先看标题和目录"
>   - **监控**："这段我没读懂——回头重读""作者这里好像有偏见"
>   - **评估**："核心论点是 X，但证据不够充分，我需要找其他来源验证"
>
> **会议中的元认知：**
>
> - 发言前自检："我是基于事实还是基于情绪在说话？"
> - 听别人发言时："他的逻辑链条完整吗？我是否因为认同结论而忽略了论证漏洞？"

## 注意事项

- 元认知**消耗更多认知资源**——不是所有事情都需要开启元认知（刷牙不需要）
- 元认知能力**可以通过训练提升**——冥想、写反思日记、学新技能都是好方法
- 过度元认知可能导致**分析瘫痪**——该行动时就行动
- 元认知是**达克效应的解药**——它让你能准确评估自己的能力水平

## 变体/相关模型

- **达克效应**：缺乏元认知导致的认知偏差
- **乔哈里视窗**：自我认知与他人认知的差异框架
- **成长型思维 (Growth Mindset)**：Dweck 的理论——相信认知能力可以发展
- **双重过程理论 (Dual Process)**：系统1/系统2 的监控工具
- **反思性实践 (Reflective Practice)**：Schön 的教育理论——实践中嵌入元认知

## 适用场景

- 学习新技能/新领域
- 复杂问题决策
- 创意工作（写作、设计、编程）
- 冲突沟通中的自我调节
- 考试/面试准备
- 领导力发展
- 错误归因与纠正
