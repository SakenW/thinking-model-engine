# 波特五力 (Porter's Five Forces)

## 概述

迈克尔·波特 1979 年提出的**行业竞争结构分析框架**，评估一个行业的**吸引力和盈利能力**。

## 五种力量

```
          ┌────────────────┐
          │   新进入者威胁    │ ← 进入壁垒
          │ Threat of New   │
          │     Entrants    │
          └───────┬────────┘
                  │
                  ▼
┌───────────┐   行业竞   ┌──────────────┐
│ 供应商议价  │ ← 争对手 → │ 客户/买家议价  │
│ 砍能力     │ Rivalry   │ 砍能力        │
│ Supplier  │ Among     │ Buyer        │
│ Power     │ Existing  │ Power        │
└───────────┘ Competitors└──────────────┘
                  │
                  ▼
          ┌────────────────┐
          │  替代品威胁      │ ← 替代压力
          │ Threat of       │
          │ Substitutes     │
          └────────────────┘
```

### 1. 行业内竞争 (Rivalry)
竞争激烈程度取决于：
- 竞争者数量和规模
- 行业增长速度（增长慢 = 抢存量 = 更激烈）
- 产品差异化程度
- 固定成本占比（高固定成本 = 价格战倾向）
- 退出壁垒

**竞争激烈 → 利润被压缩**

### 2. 新进入者威胁 (Threat of Entry)
进入壁垒越高，威胁越小：
- 规模经济 / 资金需求
- 品牌忠诚度 / 转换成本
- 政策许可 / 专利技术
- 渠道控制 / 先发优势

### 3. 替代品威胁 (Threat of Substitutes)
- 有没有其他方式满足同样需求？
- 性价比对比如何？
- 转换难度多大？

> **注意：替代品不一定是同类产品。**
> 可口可乐的替代品不只是百事可乐，还有茶、咖啡、瓶装水。

### 4. 供应商议价能力 (Supplier Power)
- 供应商集中度 vs 买方数量
- 转换供应商的成本
- 供应商前向一体化的可能性
- 该投入对供应商业务的重要性

### 5. 买家/客户议价能力 (Buyer Power)
- 买家集中度 vs 卖方数量
- 产品标准化程度（标准产品 = 易换供应商）
- 买家后向一体化的可能性
- 价格敏感度 / 转换成本

## 使用方法

1. 对每种力量打分（1-5 或高/中/低）
2. 绘制五力图
3. 找到**最关键的力量**（通常 1-2 种决定行业格局）
4. 制定策略：增强自身地位或改变行业结构

## 输出 → SWOT 的输入

- 五力分析的结果直接映射到 SWOT 的 O 和 T
