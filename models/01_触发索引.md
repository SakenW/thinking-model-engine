# 思维模型触发索引

> 你要做什么？→ 用哪些模型。按实际场景索引。

---

## 🎯 我要做决策

**选什么方案 / 去哪条路**

| 场景 | 推荐模型 |
|------|----------|
| 多个选项不知道选哪个 | 决策矩阵、二阶思维 |
| 选择有风险，怕后悔 | 10-10-10 旁观思维、安全边际 |
| 面对不确定性/极端风险 | 黑天鹅理论、Cynefin 框架 |
| 需要快速迭代决策 | OODA 循环 |
| 判断事情值不值得做 | 帕累托法则、机会成本 |
| 不确定自己够不够懂 | 能力圈、达克效应 |
| 长期 vs 短期利益冲突 | 双目标清单、沉没成本 |

## 🔍 我要分析问题

**出了问题 / 搞不清楚怎么回事**

| 场景 | 推荐模型 |
|------|----------|
| 不知道问题出在哪 | 5 Whys、问题树、莫塔5问 |
| 问题太复杂想拆开 | 5W1H、MECE 原则、第一性原理 |
| 表面问题反复出现 | 大冰山模型、反馈回路 |
| 不确定问题有多复杂 | Cynefin 框架 |
| 想从反面思考 | 逆向思维 |

## 📊 我要做战略/规划

**定方向 / 看大局**

| 场景 | 推荐模型 |
|------|----------|
| 分析行业/竞争环境 | 波特五力、PESTEL 分析、3C 战略三角 |
| 评估公司/业务组合 | BCG 矩阵、SWOT 分析、护城河理论 |
| 找增长方向 | 飞轮效应、STP 营销战略、长尾效应 |
| 设定目标 | SMART 原则、OKR、洛克目标设定理论、OGSM |
| 理解系统如何运作 | 系统思维、路径依赖、反馈回路 |

## 💡 我要做产品/做项目

**从想法到落地**

| 场景 | 推荐模型 |
|------|----------|
| 验证想法值不值得做 | MVP 最小可行产品、OODA 循环 |
| 确定功能优先级 | RICE 优先级模型、艾森豪威尔矩阵 |
| 理解用户行为 | 福格行为模型、Hook 上瘾模型、AISAS 模型 |
| 产品设计 | 设计思维、SCAMPER |
| 保障质量 | 国军标六性、PDCA 循环 |

## ✍️ 我要写方案/做汇报/演讲

**让人听懂、被说服**

| 场景 | 推荐模型 |
|------|----------|
| 写报告/邮件 | 金字塔原理、SCQA 结构表达 |
| 即兴发言/面试 | PREP 结构、STAR 法则 |
| 讲故事/说服 | 黄金圈法则、FAB 销售法则 |
| 团队讨论引导 | 六顶思考帽、头脑风暴、放大镜与缩小镜 |
| 复盘/总结 | WSN 模型、GRAI 复盘、KISS 复盘 |
| 收尾要给人留下印象 | 写在后面、峰终定律 |

## 👥 我要管团队/带人

**激励、沟通、组织**

| 场景 | 推荐模型 |
|------|----------|

## 📈 我要提高效率/管理时间

**做正确的事，正确地做事**

| 场景 | 推荐模型 |
|------|----------|
| 事情太多做不完 | 艾森豪威尔矩阵、帕累托法则、GTD |
| 总是拖延 | WOOP 防拖延、帕金森定律 |
| 需要深度工作 | 时间阻塞、心流、耶克斯-多德森定律 |
| 追求持续进步 | PDCA 循环、费曼学习法 |

## 🧠 我要理解人/理解自己

**心理学和行为**

| 场景 | 推荐模型 |
|------|----------|
| 理解为什么人不理性 | 卡尼曼双系统、前景理论、心理账户 |
| 理解偏见和误判 | 框架效应、合取谬误、沉没成本 |
| 理解社会现象 | 马太效应、沉默的螺旋、墨菲定律 |
| 自我认知 | 达克效应、元认知、乔哈里视窗 |
| 理解体验和记忆 | 峰终定律、近因效应、贝勃定律 |

## 🚀 我要创新/找灵感

**打破惯性，产生新想法**

| 场景 | 推荐模型 |
|------|----------|
| 系统性发散思维 | SCAMPER、头脑风暴、奥斯本检核表 |
| 回到本质重新构建 | 第一性原理、奥卡姆剃刀 |
| 理解创新如何传播 | 长尾效应、断裂点理论 |
| 在复杂中寻找方向 | 曼陀罗思考法、系统思维 |

## 快速组合推荐（常用套餐）

| 做什么 | 模型组合 |
|--------|----------|
| **做产品决策** | Cynefin（判断复杂度）→ MVP/OODA（验证）→ PDCA（改进）|
| **做战略规划** | PESTEL（宏观）→ 波特五力（行业）→ SWOT（自身）→ OKR（落地）|
| **做复盘** | WSN（反思）→ GRAI（结构化）→ PDCA（改进闭环）|
| **面试准备** | STAR（讲经历）→ PREP（回答问题）→ 金字塔原理（结构化表达）|
| **解决反复出现的问题** | 大冰山模型（找深层原因）→ 5 Whys（挖根因）→ 反馈回路（改系统）|
| **时间管理** | 艾森豪威尔矩阵（分类）→ 帕累托法则（聚焦）→ 时间阻塞（排日程）|
| **说服/销售** | FAB（产品价值）→ PREP（结构化表达）→ 框架效应（表述技巧）|
| **个人成长** | T型能力模型（方向）→ 能力圈（聚焦）→ 心流（状态）→ 元认知（觉察）|
