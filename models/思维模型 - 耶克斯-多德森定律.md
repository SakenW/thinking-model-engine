# 耶克斯-多德森定律 (Yerkes-Dodson Law)

## 概述

**压力与绩效呈倒 U 型曲线：适度压力提升表现，过低或过高都降低表现。**

1908 年由心理学家 Robert Yerkes 和 John Dodson 通过动物实验发现，是最早揭示"压力-绩效"关系的心理学定律。

## 核心曲线

```
绩效
  ↑
  │        ╱ 峰值
  │      ╱
  │    ╱   ← 最佳唤醒区（适度压力）
  │  ╱
  │╱
  └──────────────────→ 压力/唤醒水平
  低       适度      高
  (无聊)           (焦虑/崩溃)
```

## 关键发现

| 唤醒水平 | 状态 | 绩效 |
|----------|------|------|
| **过低** | 无聊、昏昏欲睡、缺乏动力 | ❌ 表现差 |
| **最佳区** | 适度紧张、专注、有挑战感 | ✅ 表现最佳 |
| **过高** | 焦虑、恐慌、认知过载 | ❌ 表现下降 |

### 任务复杂度的调节作用

一个容易被忽略的**关键变量**：

| 任务类型 | 最佳唤醒水平 | 原因 |
|----------|-------------|------|
| **简单/机械任务** | 较高的压力也能表现好 | 不需要太多认知资源 |
| **复杂/创意任务** | 需要较低的唤醒水平 | 高压力会占用认知带宽 |

> 做数学题和搬砖需要的压力水平不同。

## 实际应用

### 自我管理
- **感到无聊 → 主动增加挑战**：缩短截止时间、加入竞争元素、提高标准
- **感到焦虑 → 主动降低压力**：拆分任务、延长时间、先做最简单的部分
- **找到"甜点"**：那个让你"有点紧张但很有干劲"的状态

### 管理团队
- 给新手安排高压力简单任务（加速学习）
- 给专家安排低压力复杂任务（释放创造力）
- 不同人最佳区不同，不要一刀切

### 考试/演讲
- 完全放松 ≠ 最好表现（适度紧张才能发挥）
- 过度紧张 → 深呼吸、降预期、把注意力从"结果"转向"过程"

## 注意事项

- "最佳区"因人而异，也因任务而异
- 长期高压即使单次表现还行，也会导致倦怠
- 和心流模型互补：心流 ≈ 压力曲线峰值区域的体验描述

## 与其他模型的关系

- **心流**：耶克斯-多德森定律的"峰值区"就是心流状态
- **帕金森定律**：高压下工作膨胀 → 可能推过最佳区
- **马斯洛需求层次**：安全感未满足时，压力容易过高
