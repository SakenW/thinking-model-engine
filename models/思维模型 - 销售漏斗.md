# 销售漏斗 (Sales Funnel)

## 概述

销售漏斗描述潜在客户从**首次接触**到**最终成交**的完整旅程，形似一个漏斗——顶部宽（大量潜客），底部窄（少量成交）。每个阶段都有流失，优化的目标是提高各阶段的转化率。

## 漏斗结构

```
        潜在市场 (Total Addressable Market)
                   │
              ┌────▼────┐
              │  知晓    │  Awareness     ← 广告、内容、SEO
              │  1000人  │  转化率 ~40%
              └────┬────┘
                   │
              ┌────▼────┐
              │  兴趣    │  Interest      ← 白皮书、试用、Demo
              │   400人  │  转化率 ~30%
              └────┬────┘
                   │
              ┌────▼────┐
              │  考虑    │  Consideration ← 方案对比、报价
              │   120人  │  转化率 ~25%
              └────┬────┘
                   │
              ┌────▼────┐
              │  意向    │  Intent       ← 商务谈判、合同
              │    30人  │  转化率 ~33%
              └────┬────┘
                   │
              ┌────▼────┐
              │  成交    │  Purchase     ← 付款、签约
              │    10人  │
              └─────────┘
```

## 使用步骤

1. **定义漏斗各阶段** — 根据你的业务调整阶段名称和数量
2. **测量每个阶段的数量** — 用数据工具追踪（CRM、分析平台）
3. **计算阶段转化率** — 哪一步流失最严重？
4. **诊断瓶颈** — 为什么在这里流失？是流量质量还是转化能力？
5. **针对性优化** — A/B 测试、话术优化、流程改进

## 示例

> **SaaS 产品销售漏斗优化：**
>
> - 发现"考虑 → 意向"阶段转化率只有 10%（正常应该 25%+）
> - 原因：报价后客户觉得贵，没有价值锚点
> - 优化：增加竞品对比表 + ROI 计算器 + 案例研究
> - 结果：该阶段转化率提升到 22%

## 注意事项

- 漏斗不是线性的——现代用户可能**跳过阶段**或**反复回退**
- 不要只盯着底部（成交），**顶部（流量质量）** 同样重要
- 不同渠道的漏斗形状不同——SEO 漏斗长，直播带货漏斗短
- B2B 和 B2C 的漏斗深度差异很大

## 变体/相关模型

- **AISAS 模型**：互联网时代的漏斗变体（加入搜索和分享）
- **Hook 上瘾模型**：产品侧的用户留存漏斗
- **AARRR 海盗指标**：增长黑客版的漏斗框架
- **客户生命周期价值 (CLV)**：漏斗底部的延伸——成交后的长期价值

## 适用场景

- 销售流程设计与优化
- 营销预算分配（投在哪一阶段 ROI 最高）
- CRM 系统搭建
- 转化率优化 (CRO)
- 创业公司增长规划
