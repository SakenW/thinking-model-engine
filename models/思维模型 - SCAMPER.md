# SCAMPER 创新法

## 概述

**七步强制发散思维法**，通过 7 个方向的提问打破惯性思维，产生创新想法。

由 Bob Eberle 基于 Alex Osborn 的头脑风暴清单整理而成。

## 七个维度

| 字母 | 英文 | 中文 | 核心问题 |
|------|------|------|----------|
| **S** | Substitute | 替代 | 能用什么替代？材料/人员/流程/规则？ |
| **C** | Combine | 合并 | 能把什么合并在一起？功能/目的/创意？ |
| **A** | Adapt | 改造/借鉴 | 能从别的领域借鉴什么？类似的东西？ |
| **M** | Modify / Minimize / Magnify | 调整 | 改变形状/大小/颜色/频率？放大或缩小？ |
| **P** | Put to other uses | 另作他用 | 还有其他用途吗？换个场景？ |
| **E** | Eliminate | 去除 | 能去掉什么？简化？减少？ |
| **R** | Reverse / Rearrange | 重排/反转 | 反过来做？调换顺序？改变角色？ |

## 使用步骤

1. **明确问题/对象**：要改进什么产品、服务或流程
2. **逐个维度过一遍**：每个字母至少想出 3 个点子
3. **不评判先发散**：数量 > 质量，后面再筛选
4. **组合优化**：把不同维度的好想法交叉组合
5. **筛选落地**：可行性 + 影响力评估

## 示例：用 SCAMPER 改进"咖啡店"

| 维度 | 想法 |
|------|------|
| S 替代 | 用植物奶替代牛奶；用手机点单替代排队 |
| C 合并 | 咖啡 + 书吧；咖啡 + 联合办公 |
| A 借鉴 | 借鉴订阅制（月卡无限杯）；借鉴游戏化（集章换购） |
| M 调整 | Mini 杯型（试饮装）；24 小时营业 |
| P 另用 | 下午变酒吧；空间出租给小团队开会 |
| E 去除 | 去掉收银台（全移动支付）；去掉固定座位（站立式） |
| R 反转 | 客户自己冲咖啡；先喝后付（信任模式） |

## 适用场景

- 产品改进 / 新产品开发
- 流程优化
- 成本降低
- 寻找新的商业模式
- 个人项目头脑风暴

## 技巧

- **和团队一起做** — 不同人擅长不同维度
- **强制每个维度至少 3 个想法** — 第一层通常是浅的，逼自己深挖
- **最意想不到的维度往往最有价值**
