# BCG 矩阵

## 概述

波士顿咨询集团 (BCG) 1970 年提出的**业务组合分析工具**，用两个维度把业务/产品分为四类，指导资源分配。

## 两个维度

| 维度 | 含义 | 衡量方式 |
|------|------|----------|
| **市场增长率** (纵轴) | 行业/市场的吸引力 | 通常以 10% 为分界线 |
| **相对市场份额** (横轴) | 业务在行业中的竞争地位 | 对数刻度，1.0 为分界线 |

## 四象限

```
        高增长率 ↑
            │
    ┌───────┼───────┐
    │ ❓      │       │
    │ 问题    │  明星  │
    │Question │  Star │
    │        │       │
    ├───────┼───────┤→ 高份额
    │ 🐄     │       │
    │ 现金牛 │  狗    │
    │ Cow    │  Dog   │
    │        │       │
    └───────┴───────┘
          低增长率 ↓
         低份额 ←──────── 高份额
```

### ⭐ 明星 (Stars)
- **高增长 + 高份额**
- 行业领导者，需要大量投入维持地位
- **策略**：投资、保护份额 → 未来变成现金牛

### 🐄 现金牛 (Cash Cows)
- **低增长 + 高份额**
- 市场成熟，不需要大额投入，持续产生现金流
- **策略**：挤奶 — 用它的收入供养明星和问号

### ❓ 问题/问号 (Question Marks)
- **高增长 + 低份额**
- 有潜力但竞争地位弱，不确定能否做起来
- **策略**：选择性投资 — 要么做成明星，要么放弃

### 🐕 狗 (Dogs)
- **低增长 + 低份额**
- 赚不了多少钱，甚至亏损
- **策略**：剥离 / 收缩 / 卖掉（除非有战略协同价值）

## 理想演进路径

```
问题 → 明星 → 现金牛
(投入)  (成长)  (收获)
```

## 使用注意事项

- 市场增长率的分界点因行业而异（不是固定 10%）
- "狗"不一定非要砍掉 — 可能是防御性产品或战略卡位
- 不要只看财务指标，还要考虑战略协同
- 动态看：今天的明星是明天的现金牛
