# 汉密尔顿定律 (Hamilton's Rule)

## 概述

**汉密尔顿定律**（Hamilton's Rule）由演化生物学家 **威廉·汉密尔顿 (W.D. Hamilton)** 于 1964 年提出，是**社会生物学和进化心理学的核心理论**。公式：**r × B > C**——当亲缘个体获得的收益（B）乘以亲缘系数（r）大于付出者的成本（C）时，利他行为就能在进化中被自然选择保留。

> 本文件将汉密尔顿定律从生物学原意出发，延伸为一种**广义的"利他投资"决策思维模型**。

## 核心内容

### 原始公式

```
r × B > C

r = 亲缘系数 (Relatedness)
    亲子/全同胞 = 0.5
    祖孙/半同胞 = 0.25
    堂/表亲 = 0.125

B = 受益者获得的适应性收益 (Benefit)
C = 利他者付出的适应性成本 (Cost)

当 r×B > C 时 → 利他行为在进化中是有利的
```

### 为什么这解释了"无私"？

| 行为 | r | B | C | r×B > C？ |
|------|---|---|---|----------|
| 母亲为救孩子牺牲 | 0.5 | 极大（孩子一生）| 自己的命 | ✅ |
| 一个人为陌生人牺牲 | ~0 | 0 | 自己的命 | ❌ |
| 帮兄弟姐妹找工作 | 0.5 | 中等 | 小 | ✅ |

**结论**：看似"无私"的行为，从基因视角看其实是"自私"的——帮助与自己基因相近的个体，等于帮助自己的基因延续。

### 广义延伸：社会版汉密尔顿定律

| 维度 | 生物学版 | 社会版 |
|------|---------|--------|
| **r（关联度）** | 基因亲缘系数 | 情感连接、信任度、利益绑定程度 |
| **B（收益）** | 适应性收益 | 对方获得的价值/利益 |
| **C（成本）** | 适应性损失 | 你付出的时间/金钱/精力 |
| **适用范围** | 亲属间的利他行为 | 职场协作、人脉投资、社交决策 |

## 使用步骤

### 作为决策工具（值得帮吗？）

1. **评估 r（关联度）** — 对方和我有多深的连接？（血缘/信任/共同利益/情感纽带）
2. **评估 B（对方收益）** — 我的帮助能给对方带来多大价值？
3. **评估 C（我的成本）** — 我要付出什么？（时间、金钱、精力、机会成本）
4. **计算 r×B vs C** — 如果 r×B >> C，值得帮；如果 C >> r×B，应该拒绝或降低投入
5. **长期复利考量** — 今天的帮忙可能在未来以更大的 r 回报（关系加深 = r 值增加）

## 示例

> **职场中是否帮同事做额外工作？**
>
> - 同事 A：关系好、经常互相帮忙（r 高），他完成不了会影响项目进度（B 高），我只需要花 30 分钟（C 低）→ r×B >> C → **帮**
> - 同事 B：不太熟、从不回报（r 低），帮他省了 2 小时（B 中），我要花 3 小时（C 高）→ r×B < C → **婉拒或降低帮助**
>
> > **人脉投资的汉密尔顿思维**
> >
> > - 与高影响力的人建立关系（提升未来的 B 值）
> > - 定期维护深度关系（提升 r 值）
> > - 对 r 值低的关系控制 C 值（不盲目付出）
> > - 形成"利他投资组合"：分散帮助不同人，降低单点 C 值过高的风险

## 注意事项

- 汉密尔顿定律的生物学原意**不应被直接套用到人际关系**——人是复杂的，不是基因的奴隶
- 在社交中过度"计算"会显得功利——**作为后台思考工具就好，不需要处处量化**
- "关系"（r 值）不是固定的——帮助别人本身就是**提升 r 值的方式**
- 最健康的模式：**真心利他 + 聪明选择帮助对象**，而不是纯粹的交易思维

## 变体/相关模型

- **互惠利他 (Reciprocal Altruism, Trivers)**：非亲缘个体之间的"你帮我我帮你"策略
- **纳什均衡 (Nash Equilibrium)**：博弈论中的合作策略分析
- **社会交换理论 (Social Exchange Theory)**：人际关系的成本-收益分析框架
- **弱关系理论 (Strength of Weak Ties, Granovetter)**：弱关系有时比强关系更有价值——因为覆盖面更广
- **帕累托法则 (80/20)**：20% 的关系带来 80% 的价值——帮助筛选高 r×B 的对象

## 适用场景

- 职场协作中的精力分配
- 人脉经营与社交投资决策
- 团队管理中的资源分配
- 时间管理中的"帮不帮"判断
- 社交平台的关系管理
- 理解人类利他行为的演化根源
