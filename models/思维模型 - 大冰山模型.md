# 大冰山模型 (Iceberg Model)

## 概述

**你看到的只是冰山一角，真正的驱动力隐藏在水面之下。**

源自弗洛伊德的冰山理论（意识与潜意识），后被系统思维领域广泛采用，用于分析**表象与深层结构**的关系。

## 冰山结构

```
              ╱ 水面 ╲
            ╱ ─────── ╲
     ┌──────────────────┐  ← 你看到的
     │  事件/现象         │     （发生了什么）
     ├──────────────────┤
     │  行为模式          │     （反复出现的趋势）
     ├──────────────────┤
     │  结构性因素        │     （什么系统/规则导致了这些模式）
     ├──────────────────┤
     │  心智模式          │     （什么信念/假设支撑了这个结构）
     └──────────────────┘  ← 真正的根源
```

## 四个层次详解

| 层次 | 问题 | 例子 |
|------|------|------|
| **事件** | 刚刚发生了什么？ | 本月离职率突然飙升到 15% |
| **模式** | 这是孤例还是趋势？ | 过 6 个月离职率持续上升 |
| **结构** | 什么系统/规则在驱动？ | 薪酬体系 3 年未调、晋升通道阻塞、加班文化 |
| **心智模式** | 什么深层信念支撑了这个结构？ | "人才是成本不是资产""年轻员工不忠诚" |

## 核心洞察

> **大多数人的解决方案停留在事件层（"发个留任奖金"），但如果不改变深层结构和心智模式，问题会反复出现。**

| 解决层次 | 效果 | 持久性 |
|----------|------|--------|
| 事件层（应急反应） | 暂时缓解 | ❌ 很快复发 |
| 模式层（趋势干预） | 减缓趋势 | ⚠️ 部分有效 |
| 结构层（系统改造） | 根本改善 | ✅ 持久有效 |
| 心智层（信念转变） | 范式转移 | ✅✅ 深层变革 |

## 使用步骤

1. **描述事件**：具体发生了什么？（客观事实）
2. **识别模式**：这种事之前也发生过吗？频率、趋势？
3. **分析结构**：什么政策、流程、激励、资源分配导致了这个模式？
4. **追问心智**：决策者/组织深层的信念和假设是什么？
5. **设计干预**：从最深层开始制定改变方案

## 适用场景

- 组织问题诊断（为什么反复出现同样的问题）
- 个人成长（为什么我总是重复同样的错误模式）
- 社会现象分析（事件背后的结构性原因）
- 产品问题（用户反复投诉的根因）
- 团队冲突（表面矛盾 vs 深层价值观冲突）

## 与其他模型的关系

- **5 Whys**：帮助从事件层挖到结构/心智层
- **系统思维**：冰山模型是系统思维的具体应用
- **乔哈里视窗**：冰山模型关注"深度"，乔哈里关注"广度"
- **NLP 理解层次**：两者结构类似（环境→行为→能力→信念→身份）
