# 乔哈里视窗 (Johari Window)

## 概述

由心理学家 **Joseph Luft** 和 **Harry Ingham** 于 1955 年提出（名字取自两人名字的前两位：**Jo**seph + **Har**ry = Johari）。乔哈里视窗是一种**人际沟通与自我觉察模型**，通过"自己知道/不知道"和"他人知道/不知道"两个维度，将认知分为四个象限，帮助人们理解沟通中的信息差异。

## 核心内容

### 四个象限

```
                    自己知道          自己不知道
              ┌─────────────┐ ┌─────────────┐
    他人知道  │ ① 公开区    │ │ ② 盲点区    │
              │ Open Area   │ │ Blind Spot  │
              │ （双方都知道）│ │ （别人知道   │
              │             │ │   你不知道） │
              ├─────────────┤ ├─────────────┤
    他人不知道│ ③ 隐蔽区/隐藏区│ │ ④ 未知区    │
              │ Hidden Area │ │ Unknown     │
              │ （你知道    │ │ （双方都不知道）│
              │   别人不知道）│ │             │
              └─────────────┘ └─────────────┘
```

| 象限 | 名称 | 内容 | 特征 | 扩大策略 |
|------|------|------|------|---------|
| **① 公开区** Open | 自由信息 | 你的姓名、外貌、公开行为 | 沟通最高效的区域 | 保持开放，主动分享 |
| **② 盲点区** | Blind Spot | 别人能看到但你看不到的 | 你的习惯性口头禅、对他人的影响 | **寻求反馈** |
| **③ 隐藏区** | Hidden / Façade | 你知道但选择不告诉别人的 | 秘密、想法、感受 | **自我披露** |
| **④ 未知区** | Unknown | 双方都不知道的 | 潜在能力、潜意识动机 | **共同探索** |

### 核心目标：扩大公开区

```
理想方向：
        ② 盲点区 ──→ 公开区（通过寻求反馈）
        ③ 隐藏区 ──→ 公开区（通过自我披露）
        ④ 未知区 ──→ 公开区（通过共同探索）

公开区越大 → 信任越深 → 沟通成本越低 → 合作效率越高
```

## 使用步骤

1. **绘制自己的视窗** — 在四个象限里写下你能想到的内容
2. **缩小盲点区** — 主动问信任的人："我有什么你看到但我自己没意识到的特点？"
3. **适度开放隐藏区** — 在安全的关系中逐步分享更多真实的自己
4. **探索未知区** — 尝试新事物、接受挑战、做性格/能力测试
5. **动态调整** — 不同关系中的四个象限大小不同——同事 vs 密友 vs 陌生人

## 示例

> **新团队组建时的乔哈里视窗应用**
>
> - **第 1 周**：公开区很小——大家只知道彼此的名字和职位
> - **团队工作坊**：
>   - 每个人分享 3 个"别人可能不知道的事" → **缩小隐藏区**
>   - 每个人收到其他人的反馈"我注意到你是……" → **缩小盲点区**
> - **第 4 周**：公开区显著扩大——团队成员更了解彼此，协作摩擦减少
>
> > **一对一反馈场景**
> >
> > - 你觉得自己表达很清楚（公开区）
> > - 但同事觉得你说话总是带着命令语气（盲点区）
> > - 通过乔哈里视窗框架请求反馈："我想了解我在工作中的盲点，能诚实地告诉我吗？"
> > - 收到反馈后，这个信息从盲点区移到了公开区

## 注意事项

- **隐藏区不是都要打开**——适度的隐私是健康边界，过度自我披露反而让人不适
- **盲点区的反馈可能让人不舒服**——需要心理准备和安全氛围
- 不同关系的公开区大小不同——不需要对所有人都一样开放
- 乔哈里视窗是**动态的**——随着关系发展，各象限会持续变化
- 最实用的日常习惯：**定期寻求反馈 + 适度自我披露**

## 变体/相关模型

- **非暴力沟通 (NVC)**：提供安全的自我披露方法
- **360 度评估**：系统化地缩小盲点区的工具
- **DISC/MBTI 性格测评**：帮助探索未知区
- **心理安全感 (Psychological Safety)**：Edmondson 的概念——扩大公开区的组织基础
- **五遍沟通法**：缩小沟通中隐藏区和盲点区的实操工具

## 适用场景

- 团队建设与信任建立
- 绩效反馈与 1on1 谈话
- 个人品牌管理（控制信息披露）
- 冲突解决（理解信息不对称）
- 领导力发展（增加自我觉察）
- 亲密关系改善
- 面试官训练（如何通过提问探索候选人的隐藏区和未知区）
