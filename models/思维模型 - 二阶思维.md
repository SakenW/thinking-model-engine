# 二阶思维 (Second-Order Thinking)

## 概述

**不只看决策的直接后果，还要推演后果的后果（二阶、三阶…）。**

大多数人的思维停留在一阶（直接结果），真正的洞察来自**往后多想几步**。

## 三个层次

```
一阶思维（直接后果）：
"这件事很好，我们应该做。"
│
└─→ 二阶思维的追问：
   "做了之后会发生什么？别人会怎么反应？"
    │
    └─→ 三阶思维的追问：
       "那些反应又会带来什么连锁影响？"
```

## 经典案例

### 案例 1: 政府限价
- **一阶**：限价 → 消费者买得更便宜 ✅
- **二阶**：厂商利润下降 → 减产或退出市场 → 供应减少
- **三阶**：短缺 → 黑市涨价 + 质量下降 → 消费者实际更惨 ❌

### 案例 2: 公司全员加薪
- **一阶**：员工更开心、更有动力 ✅
- **二阶**：成本上升 → 可能裁员/减少招聘 / 产品涨价
- **三阶**：留下的人工作量增加 → 压力大 → 离职率上升 ❌

### 案例 3: 免费增值模式
- **一阶**：免费用户多了 → 更多人知道产品 ✅
- **二阶**：免费用户增加服务器成本 → 转化率可能下降
- **三阶**：需要精细化运营免费用户 → 或者调整产品策略

## 使用方法

### 方法 1: "然后呢？"循环
对每个决策连续问 **3 次**"然后呢？"

1. 如果我做了 X，**然后呢？**
2. 那个结果发生后，**然后呢？**
3. 再之后，**然后呢？**

### 方法 2: 利益相关者分析
从不同角色的视角推演：
- 客户会怎么反应？
- 竞争对手会怎么做？
- 团队成员会有什么感受？
- 监管机构会怎么看？

### 方法 3: 时间维度
- 即时效果 (T+0)
- 短期效果 (1 周 ~ 3 月)
- 中期效果 (3 月 ~ 1 年)
- 长期效果 (1 年以上)

## 注意事项

- **不要过度分析** — 不是每个咖啡订单都需要二阶思维
- **重大决策才值得深挖** — 招聘、投资、战略、发布
- **二阶效应可能是正的也可能是负的** — 不只是找风险
- **和逆向思维配合**：二阶思维找连锁反应，逆向思维找隐藏风险
