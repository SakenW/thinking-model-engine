# 能力圈 (Circle of Competence)

## 概述

巴菲特和芒格反复强调的投资（和生活）原则：**知道自己知道什么、不知道什么，并在能力圈内行动**。能力圈不是固定的——它可以扩大，但只能**从内向外**逐步扩展，不能跳跃。

## 三层结构

```
    ┌─────────────────────┐
    │  ❌ 圈外（不知道     │
    │     自己不知道）     │
    │   → 绝对不碰        │
    │  ┌───────────────┐  │
    │  │ ⚠️ 圈边缘       │  │
    │  │ （知道自己不太懂）│  │
    │  │ → 小心谨慎      │  │
    │  │ ┌───────────┐  │  │
    │  │ │ ✅ 圈内    │  │  │
    │  │ │（真正懂）  │  │  │
    │  │ │ → 大胆做   │  │  │
    │  │ └───────────┘  │  │
    │  └───────────────┘  │
    └─────────────────────┘
```

## 核心原则

| 原则 | 说明 |
|------|------|
| **知道边界** | 比拥有大能力圈更重要的是**知道边界在哪** |
| **圈内深耕** | 在能力圈内做到极致，比圈外浅尝辄止好得多 |
| **从内向外扩展** | 新领域必须与现有能力圈有交集，才能安全扩展 |
| **承认无知** | "我不懂"不是弱点，是智慧 |

## 使用步骤

1. **画出你当前的能力圈** — 你真正懂的领域是什么？（要诚实）
2. **标记边界** — 哪些是你"以为懂但实际不懂"的？
3. **面对决策时自问** — 这件事在我的能力圈内吗？
4. **如果在圈外** — 要么放弃，要么找懂的人帮忙，要么先学习再决策
5. **定期扩展** — 选择一个与现有能力圈相邻的领域深入学习

## 示例

> **投资决策中的能力圈：**
>
> - ✅ 圆圈内：互联网 SaaS、消费品品牌、中国房地产市场
> - ⚠️ 圈边缘：生物医药（懂一点商业逻辑，不懂技术）
> - ❌ 圈外：加密货币衍生品、量子计算、大宗商品期货
>
> → 遇到生物医药项目：可以小仓位参与，但必须找专业顾问
> → 遇到加密货币衍生品：**坚决不碰**

## 注意事项

- 能力圈**不等于兴趣圈**——你喜欢的不代表你真懂
- 能力圈**会退化**——长期不用的能力会萎缩
- 最危险的不是圈外，而是**以为自己懂但实际在圈边缘**
- 扩展能力圈需要**时间和刻意练习**，不是听几个播客就行

## 变体/相关模型

- **达克效应**：解释为什么人们容易高估自己的能力圈
- **第一性原理**：扩展能力圈的深度工具
- **护城河理论**：企业版的能力圈
- **元认知**："知道自己是否知道"的底层能力
- **邓宁-克鲁格效应**：能力圈认知偏差的学术名称

## 适用场景

- 投资决策
- 职业方向选择
- 创业赛道评估
- 技术选型
- 专家咨询（什么时候该找专家）
- 自我认知与成长规划
