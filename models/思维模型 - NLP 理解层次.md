# NLP 理解层次 (NLP Logical Levels / Dilts' Logical Levels)

## 概述

Robert Dilts 基于 NLP（神经语言程序学）提出的层次模型：人的问题和改变可以在 **6 个不同层次**上发生。越往上的层次改变，影响越大；低层次的问题往往需要到高层次才能根本解决。

## 六个层次

| 层次 | 名称 | 核心问题 | 改变难度 | 示例 |
|------|------|---------|---------|------|
| **L6** | **愿景/精神** | 为了什么？超越自我的使命是什么？ | ★★★★★ | "让每个人都能享受阅读" |
| **L5** | **身份** | 我是谁？我要成为什么样的人？ | ★★★★ | "我是一个终身学习者" |
| **L4** | **价值观/信念** | 什么对我最重要？我相信什么？ | ★★★ | "成长比舒适更重要" |
| **L3** | **能力** | 我会什么？我能做什么？ | ★★ | "我有编程和写作能力" |
| **L2** | **行为** | 我做什么？具体行动是什么？ | ★ | "我每天早上写 500 字" |
| **L1** | **环境** | 我在哪里？周围有什么？ | 最低 | "我在书房里，有安静的环境" |

## 层次关系

```
L6 愿景/精神    ← 最难改变，影响最大
    ↑
L5 身份         ← "我是谁"
    ↑
L4 价值观/信念   ← "我相信什么"
    ↑
L3 能力         ← "我会什么"
    ↑
L2 行为         ← "我做什么"
    ↑
L1 环境         ← 最容易改变，影响最小
```

**核心原则：低层次的问题，往往根源在高层。**

## 使用步骤

1. **识别问题** — 你想改变什么？
2. **定位层次** — 这个问题主要在哪个层次？
3. **向上追溯** — 如果 L2（行为）有问题，问"是什么信念（L4）在驱动？" "这是不是和我的身份（L5）有关？"
4. **从高层入手改变** — 改变了身份认知，行为自然跟着变
5. **逐层对齐** — 确保各层次之间一致（L1-L6 指向同一个方向）

## 示例

> **问题："我总是拖延，无法坚持健身"**
>
> - **L1 环境**：家里没有健身器材 → 办张健身卡（可能没用）
> - **L2 行为**：没养成运动习惯 → 设闹钟提醒（效果有限）
> - **L3 能力**：不知道怎么科学训练 → 请教练（有帮助）
> - **L4 信念**："运动是痛苦的""我天生不爱运动" → ⭐ **关键突破口**
> - **L5 身份**："我不是一个运动型的人" → ⭐ **重新定义身份**
> - **L6 愿景**：与健身无关 → 可能不需要强行健身
>
> → 真正的改变：从"我不是运动型的人"变成"我是一个精力充沛的人"，行为自然改变

## 注意事项

- **不要在低层次打转**——换了环境（L1）、定了闹钟（L2），如果身份认知（L5）没变，很快会打回原形
- 改变高层需要时间——**不是一次对话就能改变身份的**
- 不同情境下最优层次不同——**紧急问题可以快速改行为（L2），长期问题要改认知（L4-L5）**
- 层次之间是**互相影响**的，不是单向的

## 变体/相关模型

- **冰山模型 (Iceberg Model)**：类似的层次框架（事件→模式→结构→心智模式）
- **马斯洛需求层次**：从需求角度的层次模型
- **成长型思维 vs 固定型思维**：L4-L5 层次的改变
- **GROW 模型**：对话中引导层次跃迁的工具
- **教练技术 (Coaching)**：NLP 理解层次是核心框架之一

## 适用场景

- 个人成长瓶颈分析
- 职业发展困惑
- 习惯改变（为什么总是改不掉）
- 团队/组织文化诊断
- 教练对话框架
- 冲突调解（冲突往往在不同层次）
