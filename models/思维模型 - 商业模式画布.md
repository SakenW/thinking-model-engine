# 商业模式画布 (Business Model Canvas)

## 概述

Alexander Osterwalder 提出的**一页纸商业模式框架**，用 9 个构造块描述一个商业模式的运作方式。

## 九宫格

```
┌─────────────┬───────────┬─────────────┐
│  重要伙伴    │  关键业务   │  价值主张    │
│  Key        │  Key      │  Value       │
│  Partners   │  Activities│  Propositions│
├─────────────┼───────────┼─────────────┤
│  核心资源    │           │  客户关系    │
│  Resources  │ ─────────→│  Customer    │
│            │           │  Relationships│
├─────────────┤           ├─────────────┤
│             │           │  目标客户    │
│  成本结构    │           │  Customer    │
│  Cost       │           │  Segments    │
│  Structure  │           │             │
└─────────────┴───────────┴─────────────┘
                    ↕
              收入来源 Revenue Streams
```

## 九大构造块详解

### 1. 目标客户 (Customer Segments)
- 我们为谁服务？
- 大众 / 小众 / 多边平台？

### 2. 价值主张 (Value Propositions)
- 我们提供什么**核心价值**？
- 解决了什么痛点？满足了什么需求？

### 3. 渠道通路 (Channels)
- 怎么触达客户？（线上/线下/直销/渠道）

### 4. 客户关系 (Customer Relationships)
- 怎么获取、保持、发展客户？（自助/专人/社区/自动化）

### 5. 收入来源 (Revenue Streams)
- 怎么赚钱？（一次性/订阅/租赁/授权/广告/佣金）

### 6. 核心资源 (Key Resources)
- 需要什么资源才能让模式跑起来？（人/IP/实物/金融）

### 7. 关键业务 (Key Activities)
- 需要做哪些关键事情？（生产/解决问题/平台/网络）

### 8. 重要伙伴 (Key Partners)
- 谁是关键合作伙伴？为什么需要他们？（优化/降风险/规模经济）

### 9. 成本结构 (Cost Structure)
- 主要成本在哪里？（固定成本为主 vs 变动成本为主）

## 使用步骤

1. **准备**：大白板/便利贴/Miro，团队一起
2. **从价值主张开始** — 这是最核心的
3. **填客户** — 谁在买单
4. **填基础设施**（资源、活动、伙伴）
5. **填财务**（成本、收入）— 算账看是否成立
6. **审视逻辑链**：客户 → 价值 → 赚钱 → 成本 → 可持续？

## 变体

- **精益画布 (Lean Canvas)**：Ash Maurya 改版，更适合早期创业（加了"问题""解决方案""独特优势"，去掉了"伙伴"和"客户关系"）
