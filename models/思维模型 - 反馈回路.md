# 反馈回路 (Feedback Loops)

## 概述

**系统通过反馈回路自我调节——正反馈放大变化，负反馈稳定系统。**

系统思维的核心概念，存在于生物、经济、组织、工程等所有复杂系统中。

## 两种回路

### 正反馈回路（增强回路 / Reinforcing Loop）
> **越多 → 越多**（或越少 → 越少）

```
    ┌──→ 越多用户 →──┐
    │                  │
    │  越多口碑传播  ←─┘
    │
    ▼
  指数增长（或指数崩溃）
```

**特征：** 放大初始变化，导致指数增长或指数崩溃

| 正面例子 | 负面例子 |
|----------|----------|
| 网络效应（微信用户越多越好用）| 恐慌挤兑（越多人取款越恐慌）|
| 复利效应（利息生利息）| 恶性循环（越焦虑越失眠越焦虑）|
| 飞轮效应（亚马逊增长飞轮）| 贫困陷阱（越穷教育越差越穷）|

### 负反馈回路（平衡回路 / Balancing Loop）
> **偏离 → 回拉 → 恢复平衡**

```
    ┌──→ 偏差增大 →──┐
    │                   │
    │  纠正动作增强   ←─┘
    │
    ▼
  趋向稳定（恒温器原理）
```

**特征：** 抵制变化，维持系统在目标范围内

| 例子 | 机制 |
|------|------|
| 体温调节 | 太热→出汗→降温；太冷→发抖→升温 |
| 库存管理 | 库存低→补货→库存回升 |
| 价格机制 | 供不应求→涨价→需求下降→恢复平衡 |
| 微习惯 | 太累→降低难度→恢复执行→增强信心 |

## 系统中的常见模式

| 模式 | 结构 | 结果 |
|------|------|------|
| **增长极限** | 正反馈 + 增长到一定程度触发负反馈 | S 曲线增长后停滞 |
| **转移负担** | 短期负反馈（治标）削弱长期负反馈（治本） | 问题反复且恶化 |
| **成功螺旋** | 正反馈持续增强 | 复利式成长 |
| **恶性循环** | 正反馈持续恶化 | 崩溃 |
| **震荡** | 两个负反馈交替主导 | 周期性波动 |

## 使用方法

### 分析问题时
1. 画出关键变量之间的因果关系
2. 标注箭头方向（A↑→B↑ 或 A↑→B↓）
3. 找出回路：是正反馈还是负反馈？
4. 找到杠杆点：在哪里介入能改变回路方向

### 介入策略
| 目标 | 策略 |
|------|------|
| 打破恶性循环 | 切断正反馈链中的某个环节 |
| 防止增长停滞 | 提前识别负反馈触发点并消除 |
| 建立良性循环 | 设计正反馈回路的初始触发条件 |

## 注意事项

- "正反馈"不是"好的反馈"，"负反馈"不是"坏的反馈"
- 大多数真实系统是正负反馈回路的**组合**，不是单一回路
- 延迟（Delay）是反馈回路中容易忽略的关键变量——效果不会立刻显现

## 与其他模型的关系

- **系统思维**：反馈回路是系统思维的基本构件
- **飞轮效应**：正反馈回路的商业应用
- **PDCA**：负反馈回路在质量管理中的应用
- **心流**：技能与挑战的负反馈调节
- **学习型组织**：组织通过反馈回路自我进化
