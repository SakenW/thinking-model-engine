# 前景理论 (Prospect Theory)

## 概述

由 **丹尼尔·卡尼曼 (Daniel Kahneman)** 和 **阿莫斯·特沃斯基 (Amos Tversky)** 于 1979 年提出，是**行为经济学的基石**，卡尼曼因此获得 2002 年诺贝尔经济学奖。该理论推翻了传统经济学"理性人"假设，揭示了人类在不确定性决策中的真实行为模式。

## 核心内容

### 三大核心原理

```
原理 1：参考点依赖 (Reference Dependence)
  人的判断不是基于绝对值，而是相对于某个参照点
  "月入 1 万"是好是坏？取决于你之前赚多少
  
原理 2：损失厌恶 (Loss Aversion)
  损失的痛苦 ≈ 2~2.5 倍等量收益的快乐
  失去 100 元的痛苦 >> 得到 100 元的快乐
  
原理 3：敏感性递减 (Diminishing Sensitivity)
  从 0→1 万的变化感 > 从 1 万→2 万
  越远离参考点，变化越不敏感
```

### 价值函数曲线 (S-shaped Value Function)

```
    价值
      │
      │     ╱ ← 收益区：凹函数（风险厌恶）
      │   ╱
      │ ╱
─────┼────────── 结果
      │ ╲
      │   ╲ ← 损失区：凸函数（风险偏好）
      │     ╲
      │       ╲ （更陡峭！损失更痛）
```

| 区域 | 曲线形状 | 风险态度 | 含义 |
|------|---------|---------|------|
| **收益区** | 凹形 (concave) | 风险厌恶 | 确定的小收益 > 不确定的大收益 |
| **损失区** | 凸形 (convex) | 风险偏好 | 赌一把避免损失 > 接受确定的小损失 |
| **损失侧更陡** | — | — | 同等量级的损失比收益感受更强 |

### 四大效应（前景理论推导出的具体效应）

| 效应 | 说明 |
|------|------|
| **确定效应 (Certainty Effect)** | 人们过度偏爱确定的结果 |
| **反射效应 (Reflection Effect)** | 在收益端风险厌恶，在损失端风险偏好 |
| **隔离效应 (Isolation Effect)** | 忽略共同部分，只看差异部分做决策 |
| **禀赋效应 (Endowment Effect)** | 一旦拥有某物，对其估值显著提高 |

## 使用步骤

1. **识别决策场景** — 这是一个涉及收益还是损失的决策？
2. **找到参考点** — 对方的心理参照点在哪里？（不是 0，而是预期/现状/过去）
3. **预判偏差方向**：
   - 收益框架下 → 对方会风险厌恶 → 给确定选项
   - 损失框架下 → 对方会风险偏好 → 强调不行动的损失
4. **设计选项** — 用对方偏好的方式呈现选择
5. **自我纠偏** — 问自己："如果去掉情绪，纯逻辑怎么选？"

## 示例

> **医生向患者推荐手术**
>
> - **收益框架**："这个手术的成功率是 90%"
>   → 患者倾向于不做（90% 不是 100%，有风险）
> - **损失框架**："如果不做这个手术，你的死亡率会增加 10 倍"
>   → 患者倾向于做（避免巨大损失）
>
> > **投资中的前景理论陷阱**
> >
> > - 股票涨了 20% → 迅速卖出（收益区风险厌恶，落袋为安）
> > - 股票跌了 20% → 长期持有等待回本（损失区风险偏好，不愿兑现损失）
> > - 结果：**截断利润，让亏损奔跑**——完全反的投资策略

## 注意事项

- 前景理论描述的是**大多数人的平均倾向**——个体差异很大
- 参考点是**可被操纵的**——先抛出一个极端数字可以锚定后续判断
- 损失厌恶系数因文化和个体而异，一般在 1.5–2.5 之间
- 了解前景理论不是为了操纵他人，而是为了**做出更好的决策 + 识破被操纵**

## 变体/相关模型

- **卡尼曼双系统 (System 1 & System 2)**：前景理论背后的认知机制
- **框架效应 (Framing Effect)**：前景理论的直接应用表现
- **锚定效应 (Anchoring Effect)**：参考点的形成机制
- **心理账户 (Mental Accounting)**：塞勒将前景理论延伸到消费领域
- **期望效用理论 (Expected Utility Theory)**：前景理论挑战和替代的对象

## 适用场景

- 投资决策与风险管理
- 营销策略与定价设计
- 谈判与说服技巧
- 公共政策制定（如器官捐献、保险购买）
- 产品设计中的用户决策引导
- 个人决策的自我纠偏工具
